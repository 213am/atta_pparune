## ⚙️ **6. 기술 스택**

- **React**, **TypeScript**, **Vite**

- **Recoil**, **Yup**

- **qrcode.react** (QR 코드 생성)

- **Swiper** (슬라이드 UI)
- **SockJS** (웹소켓)
